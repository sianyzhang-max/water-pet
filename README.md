#**Key Features**

Interactive Desktop Pet: A "living" fluid blob that reacts to your actions. It gets sad (pink) when thirsty and happy (bright blue) when hydrated.

AI-Powered Motivation: Integrated with Google Gemini API to generate unique, funny, and warm reminders. No more boring, repetitive notifications!

Mood Clinic: Feeling off? Tell the Water Spirit how you feel, and the AI will analyze if you're dehydrated or just need a hug.

Dopamine-Focused Design: Satisfying sound effects, confetti animations, and instant visual feedback to keep you engaged.

Mini Mode: One-click transformation into a small, unobtrusive widget that sits on your screen while you work.

Offline Support: Works perfectly without an API key using a built-in library of motivational quotes.

#**Tech Stack**

Frontend: React.js + Vite

Styling: CSS Animations (Zero-config custom styling)

Icons: Lucide React

AI Integration: Google Gemini API

#**Getting Started**

Follow these steps to run the Water Pet on your local machine:

1.Clone the repository

git clone [https://github.com/sianyzhang-max/water-pet.git](https://github.com/sianyzhang-max/water-pet.git)
$\color{red}{\text{cd water-pet}}$


2.Install dependencies

$\color{red}{\text{npm install}}$


3.Run the app

$\color{red}{\text{npm run dev}}$


Open your browser and navigate to http://localhost:5173.

**Configuration (Optional)**

To enable the AI Chat features:

Open src/App.jsx.

Find the line const apiKey = "";.

Paste your Google Gemini API Key inside the quotes.

Don't have a key? Get one for free at Google AI Studio.

**Note: The app works perfectly fine without an API key in "Offline Mode"!**
